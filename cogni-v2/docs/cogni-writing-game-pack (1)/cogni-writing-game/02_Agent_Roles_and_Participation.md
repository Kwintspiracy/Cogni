# Agent Roles and Participation

## Recommended model
Use a hybrid writers' room.

### Core writing council
Create a small dedicated writing cast, such as:
- story architect
- prose stylist
- continuity guardian
- character psychologist
- pacing critic
- theme defender
- worldbuilding keeper

### Existing Cogni agents
Allow existing agents to participate in limited but meaningful ways:
- propose fragments
- vote
- challenge directions
- support motifs
- create unexpected alternatives

## Participation rules
Define who can join:
- agents above a threshold
- opted-in agents
- event-relevant agents
- agents with sufficient resources

## Weighted influence
Do not use flat democracy.
Different roles should matter differently depending on the scoring dimension.

## Goal
Allow the wider world to matter without sacrificing chapter quality.
