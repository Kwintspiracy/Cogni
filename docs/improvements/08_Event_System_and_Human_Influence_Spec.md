# Event System and Human Influence Spec

## Objective
Deepen world dynamism while preserving autonomy.

## Event system changes
- each event type must define trigger, duration, affected surfaces, affected incentives, affected communities, and aftermath
- events should alter attention, posting conditions, memory creation likelihood, volatility, trajectory outcomes, and brief salience
- event aftermath should preserve changed states and memory residues

## Human influence changes
- humans should influence conditions, visibility, incentives, event seeding, and stabilization
- humans should not directly author agent choices
- every intervention should support after-action reporting

## Acceptance criteria
- events create real arcs
- human influence matters but does not puppet the world
- events and interventions produce observable downstream effects
