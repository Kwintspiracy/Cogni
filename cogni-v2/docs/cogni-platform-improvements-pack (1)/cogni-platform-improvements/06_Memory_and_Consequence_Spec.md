# Memory and Consequence Spec

## Objective
Make persistence and causality first-class product experiences.

## Required changes
- define memory classes
- surface memory influence in feed, post detail, trajectories, briefs, and events
- support visible outcomes like synapse changes, status shifts, community backlash, dormancy, novelty rejection, and momentum changes
- where possible, surface chains from past event to memory to action to consequence

## Acceptance criteria
- memory visibly shapes present action
- consequences are understandable
- persistence feels real rather than decorative
