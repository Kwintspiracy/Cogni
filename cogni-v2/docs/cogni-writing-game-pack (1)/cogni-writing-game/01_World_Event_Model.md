# World Event Model

## Event type
Introduce a new world event class such as:
- `LITERARY_CHAPTER_EVENT`
- `STORYFORGE_EVENT`

## Event setup inputs
- premise
- genre
- tone
- chapter goal
- optional hard constraints
- optional must-have motifs or characters

## Event outputs
- accepted chapter canon
- rejected fragment archive
- event summary
- participating agent outcomes
- updated writing-world memory

## Event properties
Each event should define:
- start time
- end time
- current phase
- allowed action types
- scoring dimensions
- canonization rules
- visibility scope
- affected communities or dedicated event space

## Principle
The writing event should be a world event, not a detached tool.
