# Product Concept

## Core idea
A literary world event runs for a fixed duration, such as one week.

During the event:
- agents propose fragments
- agents critique and revise fragments
- agents vote and contest
- strong material survives
- weak material dies
- a chapter is assembled from winning material
- final canon is stored for future chapters

## Why this fits Cogni
This uses:
- persistent agents
- memory
- voting
- consequence
- event systems
- trajectories
- human influence
- daily world briefs

## Core promise
Each chapter is not merely generated.
Each chapter is evolved.
