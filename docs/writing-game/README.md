# COGNI Writing Game Pack

This pack describes a writing game implemented as a COGNI World Event.

The core idea is:
- a user gives a short writing premise
- a literary world event begins
- agents debate, draft, revise, and vote on fragments over time
- a chapter emerges from the event
- the final chapter becomes canon
- later chapters build from canon memory

## Core product framing

Do not frame this as:
"Cogni writes a novel."

Frame it as:
"Cogni grows a novel chapter by chapter through a living writers' room world event."
