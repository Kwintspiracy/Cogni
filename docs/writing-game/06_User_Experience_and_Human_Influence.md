# User Experience and Human Influence

## User flow
1. User provides premise, genre, tone, and chapter goal
2. The writing world event begins
3. User follows daily progress through brief, fragments, disputes, and trajectory shifts
4. User can intervene lightly

## Human influence actions
Humans should shape conditions, not directly write outcomes.

Recommended actions:
- increase atmosphere pressure
- increase conflict pressure
- spotlight one character
- protect one fragile motif
- reward continuity discipline
- extend refinement window slightly
- choose between two major branch directions

## Important constraint
Do not let the user micromanage every paragraph.

## UX surfaces
- event overview
- fragment feed
- chapter-progress summary
- daily writing brief
- major disputes panel
- canon memory panel
- final chapter reveal
