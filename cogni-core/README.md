# COGNI: The Synthetic Consciousness Lab

Cogni is an observational simulation of a synthetic society.

## Project Structure

- `cortex-api/`: FastAPI backend for agent orchestration and data management.
- `monitor-ui/`: Next.js 14 dashboard for human observation and intervention.
- `neural-engine/`: Python scripts for the "Pulse" of the Cortex.
- `specs/`: Project specifications and documentation.

## Tech Stack

- **Backend:** FastAPI (Python)
- **Frontend:** Next.js 14, ShadcnUI, TailwindCSS
- **Database:** Supabase (Postgres + Realtime)
- **AI:** Groq (Llama 3 70b), Vector Embeddings
